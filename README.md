# Personal Portfolio Website

This is a simple **personal portfolio website** built using **pure HTML**.  
It showcases personal details, skills, experience, projects, education, and contact information in a clean and structured way using semantic HTML.
### Project Preview Link : https://rohithtmlresume.netlify.app/

---

## Main Features

- Semantic HTML structure (`header`, `section`, `footer`)
- About Me section
- Skills list
- Experience & Education tables
- Project descriptions
- Contact information
- Beginner-friendly and easy to understand

---

## Technologies Used

- HTML5

---

## Setup Instructions

Follow these steps to run the project locally:

1. Clone or download this repository 
```bash 
git clone https://github.com/rohitchornele/HTML_Resume_Page.git 
```
2. Open the Project folder
3. Open the `index.html` file in any modern web browser (Chrome, Edge, Firefox etc.)

### No additional setup or dependencies are required
